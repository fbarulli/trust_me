Process Documentation

1. Program Initialization

The main program starts and initializes the HTTP session with retry logic and proper headers.

2. Company Processing

Companies are processed sequentially to maintain control over the scraping process. Each company's data is handled separately.

3. Star Rating Groups

For each company, reviews are processed in two groups:

  • 1-3 stars
  • 4-5 stars

4. Concurrent Page Fetching

Multiple pages within each star rating group are fetched concurrently:

  • Pages are requested simultaneously
  • Rate limiting ensures 1s delay between requests
  • Responses are saved to the file system

5. Rate Limiting

Several delay mechanisms are in place:

  • 1 second between pages
  • 2 seconds between star rating groups
  • 2 seconds between companies
